# Модерация объявлений (AvitoTech 2025)

Фронтенд-приложение, работающее поверх локального API модерации объявлений.  
Сервис ускоряет работу модератора: позволяет просматривать список объявлений, переходить к детальным карточкам, изучать историю модерации и менять статус объявления.  
Backend входит в проект и не требует каких-либо ключей доступа.

#### Запуск приложения

1. Клонируйте репозиторий:  
   `git clone https://github.com/S1ach/AvitoTech2025`
2. Перейдите в директорию `tech-int3-server` и установите зависимости:  
   `npm i`
3. Запустите backend:  
   `npm run start`  
   Сервер будет доступен по адресу `http://localhost:3001`
4. Перейдите в каталог `client`, установите зависимости:  
   `npm i`
5. Запустите фронтенд:  
   `npm run dev`  
   Приложение откроется на `http://localhost:5173`
6. Документация API расположена в файле `tech-int3-server/schema.yaml` (OpenAPI 3.0).

#### Технологии

- **React** — основа интерфейса и построения компонентов.
- **TypeScript** — типизация, контроль структуры данных и повышение предсказуемости кода.
- **Vite** — быстрый dev-сервер и сборщик с минимальными накладными расходами.
- **Material UI** — компонентная библиотека для базового интерфейса и сетки.
- **TailwindCSS** — утилитарная стилизация и кастомизация UI.
- **axios** — работа с API и обработка HTTP-запросов.
- **Zustand** — управление состоянием без лишней сложности.
- **shadcn/ui** — расширенный набор UI-компонентов для отдельных блоков.
- **Framer Motion** — анимации и плавные переходы в интерфейсе.
- **ag-grid** — работа с табличными данными (крупные списки и таблицы).
- **React Router** — маршрутизация между страницами приложения.

#### Описание функционала

1. Просмотр списка объявлений с поддержкой пагинации.
2. Отображение карточек с ключевой информацией: статус, цена, категория, дата, превью.
3. Переход к подробной странице объявления.
4. Внутри детальной страницы модератор может:
   - Изучить описание объявления
   - Просмотреть галерею изображений
   - Посмотреть данные продавца
   - Ознакомиться с историей модерации
   - Выполнить действия:
     - **Одобрить**
     - **Отклонить**
     - **Запросить изменение**
5. Состояние обновляется в реальном времени после действий модератора.
6. Страница статистики находится в разработке.
